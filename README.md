# mintmarks.fun


## Setup
### Circuits (`mintmarks_circuits`)
**Important:** This project requires **Nargo 1.0.0-beta.5** specifically, as the zkemail.nr library (v.1.0.1-beta.5) is compatible with this version. The project also uses **@aztec/bb.js@0.84.0** for Solidity verifier generation, which is the compatible version for Nargo beta.5.

#### Nargo 1.0.0-beta.5

```bash
# Install Nargo version 1.0.0-beta.5
noirup -v 1.0.0-beta.5

# Check the version
nargo --version
```
